## 1、阐述定义
<a name="60f1540b"></a>
### 1.1 高效设计和设计价值的定义
<a name="2bc237b0"></a>
#### 1）高效设计
高效能、高效率<br />相同或更短的时间里完成比其他人更多的任务，而且質量與其他人一樣或者更好设计。

<a name="0808a12f"></a>
#### 2）设计价值
人的主观愿望、需求和意识的产物<br />通過各種等值标准或交换标准來表示價值。是一種行為、信念、理想與規範的准则体系。

<a name="27035d38"></a>
### 1.2 合理界定高效和价值的范围，寻找平衡点
高效设计（以目标为导向，主重轻放）<br />有限的时间-理性（硬性，必须做的）<br />一致<br />功能<br />品牌<br />最大化设计价值-感性（弹性，应该做的）<br />创新<br />平衡<br />情感

#### 1）必须做（事不宜遲，一次到位）
估价<br />用目标、需要、回报對將要做的事進行估價。<br />把重要的工作，不能丢掉不管，不能拖延不做，不能授權給他人，第一次就做到位，它會節省我們很多的人力、物力、財力，使我們少走很多不必要的彎路。

<a name="fb447c08"></a>
#### 2）稍后做（制定計劃，儲備工作）
制定计判<br />制定計劃將把一些次要的、不紧急的工作放一放。<br />等完成重要的、緊急的工作之後，再按部就班地高效完成。

<a name="e4e93802"></a>
#### 3）授权做（統籌安排，提高效率）
最大化价值<br />將任務分發出去，做到團隊内人盡其才，オ盡其用，人事相宜，最大限度地發揮人力的作用。<br />通過優勢定位、動態調節實現科學合理的配置。

<a name="107d773d"></a>
#### 4）不做（别再折騰，减少玩弄）
估价<br />面對一個設計項目，设计和玩弄之間其實是有一條界限的。<br />當你覺得某個設計差不多可以了，接下來頂多往上面加一些奇奇怪怪的效果或者添加一些實驗階段的功能的時候，你就應該停手了。
